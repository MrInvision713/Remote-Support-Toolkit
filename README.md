# Remote-Support-Toolkit
A Remote Support Toolkit using Python and CustomTkinter to give technicians a simple interface for common troubleshooting commands. It can ping a host, perform DNS lookup, test ports, show IP configuration, flush DNS, collect system information, and export results to a log file for ticket documentation.
